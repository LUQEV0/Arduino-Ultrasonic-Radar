# Digitális Technika II. - Ultrahangos Radar Projekt

Ez a repozitórium a Digitális Technika II. tantárgy kötelező projektfeladatát tartalmazza.
A projekt egy Arduino Uno-ra épülő, szervómotorral forgatott ultrahangos radarrendszert valósít meg Tinkercad szimulációs környezetben.

## A projekt tartalma:
- `/src/radar_code.ino`: A teljes, magyar nyelvű kommentekkel ellátott forráskód.
- `projekt_dokumentacio.pdf`: A hivatalos egyetemi formátumú részletes dokumentáció.
- `Kapcsolasi_rajz.png`: A Tinkercad áramkör vizuális bekötési képe.
